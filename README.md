<h2>Functions</h2> 

<h3>Addition</h3>
<p>Example: 2+2</p>
<p>Output: 4.0</p>

<p>Example: 3-2</p>
<p>Output: 1.0</p>

<p>Example: 3*2</p>
<p>Output: 6.0</p>
